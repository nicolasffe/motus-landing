# MOTUS Landing Page

Landing page da **MOTUS** (Sporttech) construída com **Next.js (App Router)** + **TypeScript** + **Tailwind CSS**.  
Objetivo: apresentar a plataforma, destacar a camada de IA (insights e apoio à decisão) e capturar leads para **agendamento de demonstração**.

---

## Stack

- **Next.js** (App Router)
- **React + TypeScript**
- **Tailwind CSS**
